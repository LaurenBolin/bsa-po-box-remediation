# Verafin Note Template — No Physical Address Found

Use this template when no physical address could be located after searching all systems. The record must be escalated to the BSA Officer before closing.

---

## Template

```
Address remediation — [MM/DD/YYYY]. Checked Verafin and Bankway. 
Opened ImageCenter, actDataScout ([County] County), and ARCourts. 
Searched by customer name and DOB. 

No physical address located across all systems.
AI confidence score: None — no candidate address returned.

Escalated to BSA Officer for review and disposition.

— [Analyst Initials / AUTOMATED]
```

---

## Field Guidance

| Field | What to enter |
|-------|--------------|
| Date | MM/DD/YYYY format |
| County | Customer's county based on PO Box ZIP lookup |
| AI confidence score | None — no address candidate returned by any system |
| Escalation | Always required — record cannot be closed without officer disposition |

---

## BSA Officer Follow-Up Note Template

After officer review, add a follow-up note to the same record:

```
BSA Officer review — [MM/DD/YYYY].
Disposition: [Customer outreach initiated / Risk accepted — documented / Account reviewed / Other]
Follow-up action: [Description of next step or resolution]

— [Officer Initials]
```

---

## Important

Per BSA examination standards, a documented negative search is **not** a compliance failure. An **undocumented** negative search is. This note ensures the bank can demonstrate due diligence even when no address is found.
